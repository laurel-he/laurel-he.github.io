wsl的ubuntu安装docker

参考文章：https://www.jianshu.com/p/a20c2d58eaac 步骤：
（1）wsl切换到版本2
wsl --set-default-version 2 （2）查询ubuntu启动环境
wsl -l -v （3）如果 VERSION是1代表使用的WSL1
wsl --shutdown 首先关闭WSL下运行的Linux，执行： wsl --shutdown 然后执行： wsl --set-version <Linux发行版名称> 2 发行版名称就是刚刚wsl -l -v命令显示的name

cd \wsl$\Ubuntu-16.04\

(1)wsl2下ubuntu docker安装
